 # Meals America
Адаптивный многостраничный сайт, созданный для демонстрации навыков верстки и работы с DOM на реальном проекте.

## Демо
[Посмотреть проект](https://websoty.github.io/ameals/)

## Функционал

- Адаптивная верстка (desktop → mobile)
- Ручная верстка страницы меню 
- UI/UX компоненты
- Модальные окна для форм с динамической подгрузкой через fetch
- Валидация форм и всплывающее окно после отправки
- Единый хедер/футер и навигация на всех страницах

## Технологии

- HTML5
- SCSS (переменные, миксины, структура, импорт стилей и шрифтов)
- JavaScript (DOM, события, динамическое создание элементов, fetch)

## Навыки, которые демонстрирует проект

- Создание адаптивных многостраничных интерфейсов
- Реализация интерактивных элементов без сторонних библиотек
- Работа с DOM: создание, удаление, изменение элементов
- Организация кода и стилей
- Внимание к деталям и качеству пользовательского интерфейса
